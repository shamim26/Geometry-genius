1. What is DOM and what is the purpose of DOM?

DOM means Document Object Model. DOM make the copy of our html file and give it to us as an object and it's name is document. To build a dynamic website we need DOM. The purpose of DOM is to interact with web page programmatically.

2. How will you select HTML elements using DOM? Name the DOM methods.

There are many methods to select a HTML element,

  • getElementById
  • getElementsByClassName
  • getElementsByTagName
  • querySelector
  • querySelectorAll

    • 3. What is event bubble?

    In DOM when an event triggered the element handles the event and then it bubbles up to it's parent element and then it's parent's parent. This whole process goes until it reaches the root of the document and it's called event bubble.

    4. What is a callback function and why will you use it?

    A callback function passes parameter to another function. When you want a function to be executed after a specific functions execution then you need to use a callback function.
